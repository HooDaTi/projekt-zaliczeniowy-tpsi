# Projekt na "Techniki programowania systemów inteligentnych"

<p>Projekt to automat komórkowy w postaci symulacji ruchu pociągu po torach.</p>

## Uruchomienie
```bash
python src/main.py
```

## O programie
#### Program został wykonany w języku Python z wykorzytaniem biblioteki GUI tkinter i ttk, oraz programowania obiektowego (OOP). 
<p>Symulacja składa się z wyświetlacza statusu symulacji oraz ilości wykonanych kroków (na górze aplikacji).</p>
<p>Poniej znajdują się:</p>
<ul>
    <li>Tory (Track) (Ścieżka)</li>
    <li>Pociąg (Train) (Aktywna komórka)</li>
    <li>Zwrotnica (Switch)</li>
    <li>Rozwidlenie (Junction)</li>
    <li>Stacje (Stations)</li>
</ul>

<p>Na dole aplikacji znajdują się przyciski do kontrolowania symulacji.</p>
<p>Odpowiednio (od lewej):</p>
<ul>
    <li>Start</li>
    <li>Stop</li>
    <li>Reset</li>
    <li>Przełącz Zwrotnicę (Switch)</li>
</ul>
